# Hydra Global Masters

Esse aqui é o meu hub de fontes pro **Hydra Launcher**. Eu juntei os melhores sites de jogos (Repacks) em um link só pra facilitar a vida. 

São mais de **35 mil jogos** atualizados direto da fonte!

## Como usar no seu Hydra
1. Abra o seu Hydra Launcher.
2. Vá em **Configurações** > **Fontes**.
3. Copie e cole esse link lá:
        https://raw.githubusercontent.com/gatosmias1/Hydra-Global-Masters/refs/heads/main/Hydra%20Global%20Masters.json
